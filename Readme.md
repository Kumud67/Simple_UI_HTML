# 🌐 Simple Portfolio Web Page

This is a basic **portfolio website** built using **HTML and CSS**.  
It is designed to showcase personal information, skills, and projects in a clean and responsive layout.

---

## 🚀 Features

- Simple and clean UI
- Responsive design (works on mobile & desktop)
- Sections included:
  - Home
  - About
  - Skills
  - Projects
  - Contact

---

## 🛠️ Technologies Used

- HTML5
- CSS3
